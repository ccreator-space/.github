### About ccreator

We build open-source solutions designed to bridge the gap between software development and content creation. Our mission is to eliminate technical bottlenecks, allowing creators to focus on what they do best: creating.

### What We Build

*   **Generative AI Workflows:** Tools to streamline scriptwriting, ideation, and media generation using AI.
*   **Process Automation:** Seamless integrations and webhook setups to automate publishing, cross-posting, and distribution.
*   **Creator Dashboards:** Lightweight, self-hosted management panels for sponsorships, content planning, and portfolio showcases.

### Technologies

We actively develop using and integrating with the following stack:

*   MERN Stack (MongoDB, Express, React, Node.js)
*   Next.js & Angular
*   Supabase & Firebase
*   n8n

### Contributing

We welcome contributions from developers and creators alike. Whether it is a bug report, a new feature, or a documentation update, your input is valuable. Please review our contribution guidelines before submitting a pull request.
